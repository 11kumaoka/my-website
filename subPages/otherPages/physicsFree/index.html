解析

Name Explanation
多重度(multiplicity) フ1 eventにおける観測した粒子数．(観測した粒子数は当然初期粒子の衝突数に関係するため重要となる)
中心衝突度 (Centrality) フ原子核同士の衝突の際の二つの原子核の重なり具合. 最も重なっている中心衝突がなぜか中心衝突度0%.　重なり0が中心衝突度100%. 基本的には多重度から決定される．
グラウバーモデル 原子核の衝突を確率的に記述するモデル．原子核の衝突は確率的であるため，衝突の確率を計算する．
クローニン効果 高エネルギーのプロトン-プロトン衝突において、グルーオンの密度が高くなることで、粒子の生成が抑制される現象。
カラースクリーニング クォーク-グルーオンプラズマにおいて、色荷がスクリーンされる現象。これにより、ハドロンの生成が抑制される。
coalescence 。
カラーリコネクション 衝突後の粒子が再結合して新たな粒子を生成する過程。
Dead Cone Effect 質量が大きいクォークはQGPとの相互作用が少し小さくなり，軽いクォークに対してRAAが大きくなる．
R_AA, Nuclear Modification Factor (原子核効果因子) partonがQGP媒質から受ける効果をPbPb?衝突とpp衝突で比較したもの
R_CP RAAのpp衝突を周辺衝突に変えたもの. 中心衝突度の違いから効果の大きさを定量的に求める．
Net 正味の ->粒子と反粒子の差分 (Net Proton = N_p - N_pbar)
キュムラント：
v2 フーリエ級数のパラメータらしい．楕円形を示す値らしい. v1は球形, v3は三角形．(QGP（流体）ができている証拠らしい)
NBD (Negative Binominal Distribution)
HBT (Hery Brown Twist)
Yield 生成量
Cross Section 生成断面積
PDF (Parton Distribution Function) 陽子中のパートン分布
FF (Fragmentation Function) パートンからハドロンに崩壊していくプロセスを記述した関数 →Cross Section X PDF X FFによって初期状態から観測される終状態を得ることができる．一方でその逆を考えれば観測量から初期状態を決めることができるとも言える．
Hard/Soft HardはHigh pTを指すことが多い...
pQCD (purtabative QCD) 摂動論によるQCD計算 (high energy)
lattice QCD 格子計算によるQCD (low energy)
A MultiPhase Transport(AMPT) string melting scenario. 1.the initial condition(HIJING)->2.parton cascade (ZPC)->3.hadronization(ART)->4.hadronic rescatterings.
mini jet Bjorken xの小さい領域において，陽子内で小さいpTのグルーオンの数が支配的になる．このグルーオンとの散乱によって出てきたpTの小さいジェットをmini jetと呼ぶ.
Chiral Magnetic Effect (CME) QGPが渦を作って電荷が回ることによって磁場ができる．CP対称性との関係性はよくわからない...
Axion Like Particles (ALPs) アクシオンっぽい粒子.
Melting できたccbarがqgpのqと組んで溶ける
regeneration ccbarがいっぱいできると溶けてまた組み直す
dead cone effect ジェット方向にグルーオンの制動放射がいない
stopping effect 衝突系がとまる．
すり抜け効果 加速させる粒子のエネルギーをあげると構成粒子のサイズは小さくなるため，すり抜ける割合が大きくなる(不確定性原理)
chemical freeze-out 衝突初期ではクォークが単体でいられる温度であるため，さまざまなハドロンを組んだり崩壊したりすることができるが，温度の低下により，ハドロン数が一定になる．
thermal(kinetic) freeze-out 衝突系における熱が一定になる
shadowing effect 衝突系の高密度物質がspectator部分に引きづられる効果．(z(ビーム軸)方向に衝突系は傾く)
Name Explanation
QA (Quality Assurance) データの質の保証検証. runのデータにおける検出器が正常に動いているかどうかを確認し，使うデータの選別を行う.
acceptance 結果の有効範囲? 多くの場合が検出器の領域制限によるものが多いが，それらの制限の足し合わせの結果残った保証されたデータ範囲．
pT had bin シミリュレーションにおいて生成断面積の小さいhigh pTの事象を効率良く作るために，bin幅を絞って解析に必要な分の生成を行う
Responce Matrix(RM) シミュレーションで作った真値にを行，検出器の効果や背景事象の効果を含め再構成した値を列として，真値が実験でどのように変化するかの対応関係を表す行列. v_det = RM X v_true
SVD, Basian RMで行いたいことは実験値から真値を得ることであるためRMの逆行列が必要(v_true = RM^-1 X v_det)．上記で生成したRMから逆行列を求める手法としてSVDやBasianがよく使われる．
Unfolding 検出器や背景事象による影響等で変化する分布形状をシミュレーション等で補正すること.(pT分布なら横軸に対する補正)
Embedding Unfoldするとき，主に背景事象の効果を考慮するために行われる. シミュレーションで生成した粒子(truth)を実験データに埋め込み(embed)， そのtruthがどのように背景事象を含んだ上で再構成されるかを見る．
Statistic Uncertainty (統計誤差)
Systematic Uncertainty (系統誤差)

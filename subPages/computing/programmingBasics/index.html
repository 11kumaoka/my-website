shell芸

shell芸とは、シェルスクリプトやコマンドライン操作における技術やテクニックのことを指します。以下に、いくつかの例を示します。

基本構文と制御構造

コマンドリスト

コマンドが順次左から右に実行される。 
コマンド1; コマンド2; ...
コマンドが左から右に順に実行され、前のコマンドの出力（標準出力）が次のコマンドに入力（標準入力）される。 
コマンド1 | コマンド2 | ...
前のコマンドが成功した場合（終了コードが0の場合）のみ、後ろのコマンドが実行される。 
コマンド1 && コマンド2
コマンド1が正常終了以外（終了コードが0以外）だったら、コマンド2を実行されます。 
コマンド1 || コマンド2

例:「make」「make install」を順番に実行する。 途中でエラーが発生した場合、それ以降のコマンドは実行されない。このため不要なコンパイル時間の短縮になる。 

$ make && make install

コマンド置換

コマンド置換は、コマンドの実行結果を変数に代入したり、他のコマンドの引数として使用するための機能です。 「$()」を使用してコマンドを囲むことで、そのコマンドの出力が置き換えられます。 例: 現在の日時のファイルを新規作成するには、touchコマンドを使用して次のように記述できる。 

$ touch $(date +%F)
$ ls
2025-01-01

ブレース展開

「,」カンマで区切った複数の文字列を「{}」ブレース（中括弧）で囲む事で、囲んだ文字列を展開することができる。 複数のディレクトリを一度に新規作成するには、mkdirコマンドを使用して次のように記述できる。 

$ mkdir dir{1,2,3}
$ ls
dir1/  dir2/  dir3/

また、次のようにディレクトリ名に対して、部分的に中括弧を利用することもできる。 

$ mkdir dir{1..3}/subdir
$ ls
dir1/subdir/  dir2/subdir/  dir3/subdir/

変数展開

変数の値を操作するための機能。以下にいくつかの例を示す。

$ path=/home/user/file.txt
$ echo ${path%.*}     # 拡張子除去
$ echo ${path##*/}    # ファイル名取得

正規表現

正規表現は、文字列のパターンを定義するための強力なツール。特に、テキスト処理やデータ抽出に役立つ。 以下に、基本的な正規表現の構文と例を示す。

  • []: 任意の1文字
  • [^]: 指定以外の1文字
  • .: 任意の1文字
  • *: 0回以上繰り返し
  • ^: 行頭
  • $: 行末
例えば、ファイル名から拡張子を除去するには、以下のようにsedコマンドを使用します。 
  $ echo "file.txt" | sed 's/\.[^.]*$//'

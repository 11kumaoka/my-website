shellscriptの書き方

shellscriptは、LinuxやMacOSのターミナルで使われるスクリプト言語です。基本的な構文を理解しておくと、日常的なタスクの自動化やデータ処理が効率的に行えます。 以下に、基本的な構文とよく使うコマンドを示します。

option	example	mean
-eq	a -eq b	a=bならtrue
-ne	a -nq b	a not = bならtrue
-lt	a -lt b	abならtrue
-ge	a -ge b	a=>bならtrue

shell芸

shell芸とは、シェルスクリプトやコマンドライン操作における技術やテクニックのことを指します。以下に、いくつかの例を示します。

基本構文と制御構造

パイプライン

コマンド1 | コマンド2

コマンドリスト

  コマンド1 ; コマンド2
  コマンド1 && コマンド2
  コマンド1 || コマンド2

コマンド置換

  $ touch $(date +%F)

ブレース展開

  $ mkdir dir{1,2,3}

変数展開

$ path=/home/user/file.txt
$ echo ${path%.*}     # 拡張子除去
$ echo ${path##*/}    # ファイル名取得

ループ・条件分岐

for文

  $ for i in {1..5}; do echo $i; done

while文

  $ find . -type f | while read f; do wc -l "$f"; done

if文

if 条件; then
  実行内容
elif 別条件; then
  実行内容
else
  実行内容
fi

正規表現

  • []: 任意の1文字
  • [^]: 指定以外の1文字
  • .: 任意の1文字
  • *: 0回以上繰り返し
  • ^: 行頭
  • $: 行末
  $ echo "file.txt" | sed 's/\.[^.]*$//'

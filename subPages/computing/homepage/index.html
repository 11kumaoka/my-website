command description
html HTML文書のルート要素
head メタ情報（タイトル、CSSなど）を記述
body ページに表示される実際の内容
title ブラウザタブのタイトル
h1〜h6 見出しタグ（h1が最大）
p 段落
a リンク（href属性）
img 画像（src属性、alt属性）
br 改行
hr 水平線
div ブロック要素。構造やスタイルの単位
span インライン要素。部分装飾などに使用
ul 順不同リスト
ol 番号付きリスト
li リスト項目
table 表全体の要素
tr 表の行
td 表のセル（列）
th 表の見出しセル
form 入力フォーム全体
input テキスト入力やボタンなど
textarea 複数行のテキスト入力欄
button クリック可能なボタン
label フォーム入力項目のラベル
select ドロップダウンメニュー
option select の中の選択肢
link 外部CSSやファビコンを読み込む
script JavaScriptの記述・読み込み
meta ページ情報（文字コード、SEOなど）
style HTML内にCSSを直接書く
header ページ上部（ナビゲーションやタイトル）
footer ページ下部（著作権やリンク）
section 意味のあるセクションのまとまり
article 独立したコンテンツ（ブログ記事など）
nav ナビゲーションメニューの領域
main ページの主コンテンツ
aside 補足的なコンテンツ（サイドバーなど）
<!-- --> コメント（画面には表示されない）

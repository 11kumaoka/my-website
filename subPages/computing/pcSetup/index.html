  User user_at_server
  HostName server.example.com
  IdentityFile ~/.ssh/id_rsa_for_server

VS code

VScodeはVisual Studioと違い統合開発環境(IDE)ではなく単なるEditorです. ただいろんな機能(TeX, Git連携, SSH, Docker, etc)をダウンロードして追加していくことは可能です．

個人的なVScodeのメリット エディタとターミナルを１画面で操作できる キーバインドの設定がやりやすい。(vim形式やEmacs形式なども選べるし、自分で追加することもできる) 色々な拡張機能がある。 個人的に便利だと思う部分を以下の画像に簡単にまとめました． 私が設定した色やkeybindは( Color Syintax, KeyMap setting)をダウンロードして左下の歯車マークのところで読み込ませれば使えます． (範囲選択が私の好みのやり方ではできないことぐらいです．) VS codeでのTeXのやり方: https://qiita.com/rainbartown/items/d7718f12d71e688f3573 インストール方法　Mac: 1. HomeBrew がインストールされていない場合：まずは下のページからインストール 2. 当然VScodeのインストール brew install visual-studio-code --cask 3. LaTeX のインストール brew install mactex-no-gui --cask

= VSCodeの詳細な設定方法 = 右下の歯車のマークからSettingsをポチってjsonファイルを編集すればOKです。 真ん中の画面にあるようにGUIでも設定可能ですが、便利なjsonファイルをネットで見つけてコピペするのが無難です。 キーバインドも同様にKeyboard Shortcutsから設定できます。 私の設定は上にあるリンクからファイルをダウンロードできます。（いろいろと緑色になります。）
